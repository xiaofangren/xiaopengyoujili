# 多家庭隔离 + 一键切换家庭成员

## 更新了什么

### 1. 家庭隔离

**问题：** 之前所有用户的任务和奖励都是通用的，张三创建的任务李四也能看到，多个家庭混在一起很乱。

**解决：** 引入**家庭码**机制。同一个家庭码的人共享任务和奖励，不同家庭完全隔离。

### 2. 一键切换家庭成员

**问题：** 家里有多个小孩时，切换账号要重新输入昵称登录，很麻烦。

**解决：** 「我的」页面点击头像，弹出家庭成员列表，点一下就能切换。切换后积分、完成任务数自动切换。

### 3. 去掉了角色选择

**问题：** 登录时选「家长」或「小孩」没实际意义，增加了操作步骤。

**解决：** 登录页不再显示角色选项，直接输入昵称登录即可。

## 使用方法

### 首次使用（创建一个新家庭）

1. 打开 App，输入昵称，点击「开始冒险」
2. 系统会自动创建一个新家庭，并生成一个**家庭码**
3. 去「我的」页面，头像下方可以看到家庭码，点 📋 复制

### 家人加入（同家庭的人）

1. 新设备上打开 App，输入昵称
2. 在「家庭码」输入框里填上上面复制的家庭码
3. 点击登录，自动加入同一家庭，共享任务和奖励

### 切换家庭成员

1. 去「我的」页面
2. 点击顶部头像卡片
3. 弹出家庭成员列表，点击要切换的人
4. 一键切换，积分和进度自动更新

### 添加新的家庭成员

1. 在家庭成员切换弹窗中，点击「＋ 添加成员」
2. 输入昵称，点击确定
3. 新成员自动加入当前家庭，无需家庭码

### 查看家庭码

1. 去「我的」页面
2. 头像卡片下方直接显示家庭码
3. 也可以切换到「设置」标签页查看

## 旧数据处理

### 历史数据问题

**升级前创建的任务和奖励**（没有家庭码的旧数据）对所有用户可见，不属于任何家庭。张三删除一个旧任务，李四那边也会消失。

**新创建的任务和奖励**自动带上家庭码，只在本家庭内可见，不会串到其他家庭，删除也只影响自己家。

### 认领旧数据

如果你想把旧数据归到自己家庭名下，可以在「我的」→「设置」→「认领旧数据」一键操作：

1. 点击「认领旧数据」
2. 系统查出所有无主的旧任务和旧奖励
3. 批量打上当前家庭的家庭码
4. 认领后这些数据只归你家，其他家庭再也看不到
5. 认领完成后按钮自动消失（没有旧数据就不显示）

### 什么时候需要认领

- 如果你家有多个设备/多个人使用，且之前创建过任务和奖励 → **建议认领**，避免数据串到其他家庭
- 如果刚下载还未使用过 → 无需操作，新数据自动有家庭码
- 如果旧数据已经不需要了 → 可以直接手动删除，不用认领

## 常见问题

**Q：不同家庭之间会看到对方的任务吗？**
A：新创建的任务和奖励完全隔离。旧数据（升级前创建的）需要认领后才归自家所有。

**Q：切换家庭成员会影响积分吗？**
A：每个家庭成员有自己的独立积分、完成任务数、累计获得/消耗，切换后自动切换。

**Q：家庭码忘了怎么办？**
A：去「我的」页面，头像下方直接显示家庭码。

**Q：一个小孩可以在多个家庭吗？**
A：不能，一个账号只能属于一个家庭。
